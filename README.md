# Phần mềm phát nhạc và quản lý danh sách phát - Trịnh Xuân Quyền - K235480106059

## Giới thiệu

Đây là bài tập lớn môn **Lập trình Python** với đề tài **Phần mềm phát nhạc và quản lý danh sách phát (Media Player)**.

Chương trình được xây dựng bằng Python, sử dụng Tkinter để thiết kế giao diện người dùng, pygame-ce để phát nhạc, SQLite để lưu trữ dữ liệu và API để lấy lời bài hát cùng ảnh bìa album.
